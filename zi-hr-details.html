AI-Built HR Management System

ZI HR Portal is a comprehensive, full-stack human resource management system designed to handle every aspect of HR operations within an organization. Built entirely using Claude AI and Claude Code, the project demonstrates the power of AI-assisted software development — from initial architecture and database design through frontend UI components and backend API implementation.

The application features a modern React frontend with an intuitive dashboard, role-based access control, and a responsive design that works across devices. The Node.js backend provides RESTful APIs for all HR modules, ensuring scalability and maintainability.

Tech Highlights

  • Frontend: React with modern hooks and context API
  • Backend: Node.js + Express REST API
  • Database: MongoDB with Mongoose ODM
  • Auth: JWT-based authentication & RBAC
  • AI: 100% developed with Claude AI / Claude Code
